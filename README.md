# NotePad – MERN CRUD Note Taking App

A simple full-stack MERN note-taking application that allows users to create, read, update, and delete notes with a responsive user interface.

---

## Live Demo

https://notepad-mern-iw32.onrender.com

---

## Features

* Create, edit, and delete notes (CRUD)
* Responsive UI built with React and TailwindCSS
* Upstash Redis rate limiting for API abuse protection
* CORS configured for secure client–server communication
* MongoDB for persistent note storage
* Deployed on Render

---

## Tech Stack

### Frontend

* React
* TailwindCSS
* Axios

### Backend

* Node.js
* Express.js
* MongoDB with Mongoose
* Upstash Redis for rate limiting
* CORS

### Deployment

* Render (Frontend and Backend)

---

## CRUD Functionality

| Operation | Description           |
| --------- | --------------------- |
| Create    | Add a new note        |
| Read      | Fetch all notes       |
| Update    | Edit an existing note |
| Delete    | Remove a note         |

All operations are handled via RESTful API endpoints.

---

## Deployment (Render)

* Backend deployed as a Web Service
* Frontend deployed as a Static Site
* Environment variables configured in the Render dashboard
